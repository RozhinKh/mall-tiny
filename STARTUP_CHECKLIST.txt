SECTION 7: TROUBLESHOOTING
================================================================================

PROBLEM: "Unsupported major.minor version 61"
  Solution: Verify Java 17 is active
  - Command: java -version (must show 17.x.x)
  - Fix: export JAVA_HOME=/path/to/java17
  - Re-run: mvn clean package

PROBLEM: "Connection refused" (MySQL)
  Solution: Start MySQL server
  - macOS: brew services start mysql
  - Linux: sudo service mysql start
  - Docker: docker run -d -p 3306:3306 mysql:8.0

PROBLEM: "Cannot get a connection" (Redis)
  Solution: Start Redis or configure to skip
  - macOS: brew services start redis
  - Linux: sudo service redis-server start
  - Docker: docker run -d -p 6379:6379 redis:latest
  - OR disable Redis in application-dev.yml

PROBLEM: "Address already in use: bind"
  Solution: Use different port
  - Command: mvn spring-boot:run -Dspring-boot.run.arguments="--server.port=8081"
  - OR kill process using 8080: lsof -i :8080, then kill -9 <PID>

PROBLEM: "No such file or directory" (SQL scripts)
  Solution: Execute from project root directory
  - Command: pwd (should show ...mall-tiny)
  - If not correct: cd /path/to/mall-tiny

PROBLEM: Slow startup (> 60 seconds)
  Solution: Check system resources
  - Increase JVM heap: mvn spring-boot:run -Dspring-boot.run.jvmArguments="-Xmx1024M"
  - Check disk I/O: iostat 1
  - Check memory: free -m (should have > 1 GB available)
